Glossary

Problem

A problem that has been recognised through the Collections and Observations processes.

Problem Name

The name of the problem, beginning to categorise the problem within Solutions.

Collection Name

The name of the Collection collected, previously, for that person or place.

Observation Count

What count of Observation that was for that person or place.

Problem Collection

The Collection that is containing the Problem.

Cause

The cause of the Problem.

Result

The result of the Problem.
